# BYOK Chat

Bring Your Own Key — a multi-provider AI chat client built with Next.js.

## Features

- **Multi-provider** — OpenAI, Anthropic, Google, and more
- **Spaces** — organize chats with custom instructions per space
- **Branching** — fork conversations at any message
- **Vision & image generation** support
- **BYOK** — use your own API keys, nothing stored server-side

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Add your API key in Settings to start chatting.
